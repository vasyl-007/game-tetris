# game-tetris
TETRIS - a favourite game for generations since 1975 year.
   Rules of the Game:
      (1) to move "right" - press an arrow "right";
      (2) to move "left" - press an arrow "left";
      (3) to rotate figure - press an arrow "up";
      (4) to speed up - press an arrow "down".
   
 Good luck!
